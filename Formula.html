Excel Basic Formulas

IF with AND

=IF(AND(A1>50, B1>50), "Pass", "Fail")

Jab dono conditions true ho tab "Pass" dikhata hai, warna "Fail". Use hota hai jab aapko multiple shartein check karni ho.

IF with OR

=IF(OR(A1>50, B1>50), "Pass", "Fail")

Kisi bhi ek condition true ho to "Pass", warna "Fail". Use hota hai jab ek bhi condition sahi ho tab result chahiye.

Nested IF

=IF(A1>=90, "A", IF(A1>=80, "B", IF(A1>=70, "C", "Fail")))

Multiple conditions ke basis par result dene ke liye. Example: Grades dene ke liye.

Concatenate

=CONCATENATE(A1, " ", B1)

Do ya zyada text values ko jodne ke liye. Example: First name + Last name.

LEFT

=LEFT(A1, 3)

Cell ke starting ke 3 characters return karta hai. Example: Code ka prefix nikalna.

RIGHT

=RIGHT(A1, 4)

Cell ke last 4 characters return karta hai. Example: File extension ya last digits.

MID

=MID(A1, 2, 3)

2nd character se start karke 3 characters extract karta hai. Use hota hai string ke beech ka part nikalne me.

LEN

=LEN(A1)

Cell me total characters count karta hai. Use hota hai data length verify karne me.

TRIM

=TRIM(A1)

Extra spaces hata deta hai. Useful jab copy-paste ki wajah se extra spaces aa jaate hain.

TODAY

=TODAY()

Aaj ki tareekh return karta hai. Date tracking ke liye useful hai.

NOW

=NOW()

Aaj ki date aur time return karta hai. Real-time data tracking ke liye use hota hai.

VLOOKUP

=VLOOKUP(101, A2:C10, 2, FALSE)

101 ID dhoondhkar uska matching value doosre column se laata hai. Lookup tables me use hota hai.

HLOOKUP

=HLOOKUP("Maths", A1:E3, 2, FALSE)

Horizontal rows me value search karta hai. Tabular data me row-wise search ke liye.

COUNTIF

=COUNTIF(A1:A10, ">50")

Jo values 50 se badi hain unka count karta hai. Conditions ke basis par counting ke liye use hota hai.

SUMIF

=SUMIF(A1:A10, ">50")

Sirf unhi values ka sum karega jo condition (50 se badi) fulfill karti hain.

ROUND

=ROUND(A1, 2)

Value ko 2 decimal places tak round karta hai. Financial calculations me use hota hai.

ROUNDUP

=ROUNDUP(A1, 0)

Value ko upar ki taraf round karta hai. Use hota hai jab hamesha bada number chahiye ho.

ROUNDDOWN

=ROUNDDOWN(A1, 0)

Value ko neeche ki taraf round karta hai. Use hota hai jab hamesha chhota number chahiye ho.

COUNTA

=COUNTA(A1:A10)

Non-empty cells ko count karta hai. Jab aapko data-entry cells ginni ho.

COUNTBLANK

=COUNTBLANK(A1:A10)

Khaali cells ko count karta hai. Use hota hai missing entries track karne ke liye.

IFERROR

=IFERROR(A1/B1, "Error")

Agar error aaye to default message ya value show karta hai. Use hota hai safe calculation ke liye.

ISNUMBER

=ISNUMBER(A1)

Dekhne ke liye ki cell me number hai ya nahi (TRUE/FALSE return karta hai).

ISTEXT

=ISTEXT(A1)

Check karta hai ki value text hai ya nahi.

UPPER

=UPPER(A1)

Text ko uppercase (capital) me convert karta hai.

LOWER

=LOWER(A1)

Text ko lowercase me convert karta hai.

PROPER

=PROPER(A1)

Text ke har word ka first letter capital banata hai.

REPT

=REPT("*", 5)

Ek character ya text ko repeat karta hai given number of times.

SUBSTITUTE

=SUBSTITUTE(A1, "old", "new")

Text me kisi word ya character ko replace karta hai.

TEXT

=TEXT(A1, "dd/mm/yyyy")

Date ya number ko specific format me convert karta hai.

INT

=INT(A1)

Number ka integer part return karta hai (decimal hata kar).

MOD

=MOD(A1, B1)

Division ke baad remainder return karta hai.

ABS

=ABS(A1)

Number ka absolute value return karta hai (negative ko positive banata hai).

RAND

=RAND()

0 aur 1 ke beech random number deta hai. Simulation ke liye use hota hai.

RANDBETWEEN

=RANDBETWEEN(1, 100)

1 se 100 ke beech random number deta hai.

DATE

=DATE(2025, 4, 30)

Year, month aur day se date banata hai.

YEAR

=YEAR(A1)

Date se year extract karta hai.

MONTH

=MONTH(A1)

Date se month extract karta hai.

DAY

=DAY(A1)

Date se day extract karta hai.

NOW with Text

=TEXT(NOW(), "hh:mm:ss AM/PM")

Current time ko formatted text me dikhata hai.

DAYS

=DAYS(A2, A1)

Do dates ke beech kitne din ka gap hai wo batata hai.
